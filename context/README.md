# Context Files

Drop notes, logs, config snippets, or any other info about issues here and open a pull request.

## Workflow

1. Save your info into **`new/`** — plain `.txt` or `.md` files work great
2. Name it descriptively, e.g. `ftp-error-log-2026-06-01.txt`
3. Commit and push to a new branch, then open a pull request
4. Once the issue is resolved we'll move it to **`processed/`**

## What to include

- Error messages or log output from `allsky_windy.py`
- Windy page behaviour notes (e.g. "image shows correct but is 2 hours old")
- AllSky config details relevant to the issue
- Anything else that helps explain the problem

## Folders

| Folder | Purpose |
|--------|---------|
| `new/` | Drop new context files here |
| `processed/` | Resolved issues — kept for reference |
