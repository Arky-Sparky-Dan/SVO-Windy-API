# Screenshots

Drop screenshots of issues here and open a pull request so we can see what needs fixing.

## Workflow

1. Take a screenshot of the problem
2. Save it into **`new/`** — name it something descriptive, e.g. `windy-image-not-updating-2026-06-01.png`
3. Commit and push to a new branch, then open a pull request
4. Once the issue is resolved we'll move it to **`processed/`**

## Folders

| Folder | Purpose |
|--------|---------|
| `new/` | Drop new screenshots here |
| `processed/` | Resolved issues — kept for reference |
